About

Event Horizon is an event management platform designed to help communities publish events, manage registrations, and build integrations using a simple API.

What you can do here

  • Create and share events
  • Collect registrations and custom form answers
  • Use OAuth2 or API keys to build integrations

Open source

The project is open source. You can explore the code, report issues, or contribute improvements.
